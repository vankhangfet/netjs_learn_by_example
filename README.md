## Installation

```bash
$ npm install
```

Note: In this project, we need to install package as below:

1. To install PostgreSQL package, use the below command
```bash
npm install pg --save
```
2. To support Open API. Please see below: 
https://docs.nestjs.com/openapi/introduction
```bash 
$ npm install --save @nestjs/swagger swagger-ui-express
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
